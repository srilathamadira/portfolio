Project 2: TIC-TAC-TOE

Description

A web-based Tic Tac Toe game that allows two players to play against each other. The game is built using HTML, CSS, and JavaScript.

Design:

  • Simple and clean design for easy user interaction.
  • 3x3 grid game board with square cells.
  • Player marks (X and O) displayed in the cells.
  • Game status and winner display area.
  • Reset button

Functionality:

  1. Player 1 (X) makes the first move by clicking on an empty square.
  2. Player 2 (O) makes their move by clicking on an empty square.
  3. The game checks for wins, draws, and invalid moves after each player's turn.
  4. If a player wins, the game displays the winner and ends.
  5. If the game is a draw, the game displays a draw message.
  6. Players can reset the game to start a new game.
